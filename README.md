#  Market Forecaster

An AI-powered **stock & crypto forecasting dashboard** built with Streamlit.

It combines:
- Technical indicators 
- Machine learning forecasting   
- Financial data APIs   
- Interactive visual analytics

##  Features

###  Market Data
- Real-time stock & crypto prices
- Historical data via Yahoo Finance

###  AI Forecasting Models
- ARIMa forecasting
- Prophet forecasting
- LSTM / GRU (optional advanced mode)

### Technical Indicators
- RSI (Relative Strength Index)
- MACD
- Moving Averages
- Bollinger Bands

###  Trading Signals
- Buy / Sell / Hold signals
- Trend detection logic
- Confidence scoring

###  Backtesting Engine
- Strategy performance evaluation
- Buy & Hold comparison
